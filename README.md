# Human Activity Recognition

A Human Activity Recognition (HAR) pipeline developed using wearable sensor data, signal processing techniques and machine learning.

The project explores both handcrafted feature engineering and deep feature extraction through pretrained embeddings, comparing different data representations and evaluation strategies.

## Features

- Sensor data preprocessing
- Outlier detection
- Signal processing
- Feature engineering
- FFT analysis
- PCA dimensionality reduction
- ReliefF feature selection
- SMOTE oversampling
- Deep embedding extraction (HARNet5)
- Manual K-Nearest Neighbors implementation
- Within-subject and Between-subject evaluation
- Statistical significance analysis
